# Google 第三方登入

## 安裝及運行

1. 建立環境變數

請自行至 [Google Developers Console](https://console.developers.google.com/) 申請 OAuth 2.0 憑證，並將 .env.sample 轉為 `.env` 檔案，且按照需求填入。

2. 終端機輸入：

```
npm install
npm start
```

並直接開啟服務 `{host}/index.html` 即可體驗第三方登入的功能。
